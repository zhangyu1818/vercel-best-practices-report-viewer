# Vercel Best Practices Report Viewer

A visualizer for the JSON reports generated by the `vercel-react-best-practices-report` skill.

## Usage

1. **Start the Application**
   Run the development server:
   ```bash
   pnpm dev
   ```

2. **Load a Report**
   Drag and drop the `reports.json` file (generated by the analysis skill) onto the application window.

3. **Explore the Results**
   - **Dashboard**: View the overall Health Score and Total Violations count.
   - **Detailed Analysis**: Scroll through the list of violations.
   - **Code Diffs**: Review the "Original Code" vs. "Suggested Fix" with syntax highlighting.

4. **Customize View**
   Toggle between Light and Dark mode using the theme switcher.
